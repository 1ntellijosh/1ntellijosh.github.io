StarFighter

https://1ntellijosh.github.io/

A 2d Space Shooter I made. To win you must get through level 10, but the game will continue to go and get harder with more and more enemies. If you get 1000 points without dying, your gun upgrades and will again if you get another 1000. Run out of health, game over. Progress a level and you are granted one health bar.
